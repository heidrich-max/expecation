# Expecation.de

The official landing page for Expecation.de. Built with a focus on high standards, premium aesthetics, and modern web performance.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with Variable tokens, Flexbox/Grid, and Dark Mode.
- **JavaScript**: Vanilla JS for animations and interactivity.
- **Assets**: Stylized SVG map.

## Setup
Simply open `index.html` in any modern browser or host it via:
- GitHub Pages
- **Plesk** (standard static hosting)
- Vercel / Netlify

## Repository
Managed at: [github.com/heidrich-max/expecation](https://github.com/heidrich-max/expecation)

## Design Principles
- **Glassmorphism**: Subtle backgrounds with blur filters.
- **Premium Dark Mode**: Deep backgrounds with gold accents.
- **Motion**: Subtle reveal animations on scroll.
